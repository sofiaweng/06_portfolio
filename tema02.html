TEMA 02

grundlæggende web

I dette temaforløb lærte vi om grundlæggende web, herunder opstart af html sider, css, responsivt design, flex og grid. Samtidigt lærte vi om designprincipper, gestaltlovene, farver og fonte. Temaet var med til at give en basisforståelse over hvordan man starter en hjemmeside op og hvilke måder, man kan gøre designet mere responsiv.

I temaforløbet fik vi til opgave, at lave en hjemmeside med tekst, billeder og layoutdiagram, vi fik givet. Ud fra det, skulle vi lave hjemmesiden, hvor vi selv fik lov til at vælge farver og fonts. Opgaven er med til at belyse, de ting vi fik lært i forløbet, herunder responsivt design ved hjælp af blandt andet grid og flex i css og designprincipper.
